Generación de Horarios

El proyecto consiste en generar el horario óptimo de un estudiante de acuerdo a diferentes aspectos. Dichos aspectos son: distancia entre edificios, tiempo libre entre clases, profesores, hora de inicio de clases, hora de finalización de clases y días libres.

La aplicación hace uso de algoritmos genéticos para hallar el horario óptimo de acuerdo a los criterios dados por el usuario.
